# Dotfiles

## Installation

1. Run `./install.sh` to install all configs
2. Or run `./install.sh nvim zsh wezterm opencode aerospace` for specific packages

The script installs `stow` with Homebrew if it is missing.

## Packages

- `nvim`
- `zsh`
- `wezterm`
- `opencode`
- `aerospace`

## Manual stow

If you prefer manual control:

1. Install GNU Stow
2. Run `stow nvim`
3. Run `stow zsh`
4. Run `stow wezterm`
5. Run `stow opencode`
6. Run `stow aerospace`
